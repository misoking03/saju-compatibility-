# 사주 궁합 분석 앱

친구들의 사주 일간을 기반으로 궁합을 분석하고 시각화하는 웹 애플리케이션입니다.

## 기능

- 친구 정보 입력 (이름, 생년월일, 양력/음력 선택)
- 사주 일간 자동 계산
- 두 사람 간의 궁합 분석 (5단계: 천생연분, 좋음, 보통, 그닥, 파국)
- 네트워크 그래프로 시각화
- 관계 필터링 (전체/천생연분/파국)
- 상세 궁합 정보 팝업

## 설치 및 실행

```bash
npm install
npm run dev
```

## 기술 스택

- React
- Vite
- react-force-graph-2d (네트워크 그래프 시각화)
